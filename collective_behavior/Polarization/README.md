# Polarization

This folder contains the code to replicate the polarization experiment from the main text of the paper.

## Contents

### `polarization.py`
Main simulation script for this experiment.

It:
- Initializes the agent-based system
- Runs the time evolution of the flock
- Computes collective metrics
- Stores trajectories and weights evolution

Outputs:

- `polarization_history.npy`  
  Time series of the polarization metric

- `weights_history.npy`  
  Time evolution of the interaction weights for all agents

- `boids_trajectories.pkl`  
  Position trajectories of all agents

- `boids_vel_trajectories.pkl`  
  Velocity trajectories of all agents

---

### `plots.py`

- Loads simulation outputs generated by `polarization.py`
