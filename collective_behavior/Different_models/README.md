# Different Models

This folder contains the code for the experiments with informed agents implementing our policy composition model and uninformed agents implementing different models from the literature.

## Overview

The folder includes three simulation scenarios:

- `Couzin followers/` – Uninformed agents follow the Couzin model [Couzin et al., 2002]
- `Cucker-Smale followers/` – Uninformed agents follow the Cucker-Smale model [Cucker and Smale, 2007]
- `Vicsek followers/` – Uninformed agents follow the Vicsek model [Vicsek et al., 1995]

In all cases, the informed agents use the policy composition dynamics and are driven toward a fixed goal destination.

---

## Contents

Each subfolder contains two scripts:

### Main simulation script

Depending on the subfolder, this is one of:

- `couzin_followers.py`
- `cs_followers.py`
- `vicsek_followers.py`

It:
- Initializes the mixed-agent flocking system
- Assigns the first `informed_boids` agents as informed agents
- Evolves informed agents using the policy composition model
- Evolves uninformed agents using the literature model corresponding to the subfolder.
- Computes and stores collective metrics
- Stores trajectories and weights evolution

Outputs:

- `weights_history.npy`  
  Time evolution of the policy weights for all agents

- `polarization_history.npy`  
  Time series of the polarization metric

- `milling_history.npy`  
  Time series of the milling metric

- `distance_history.npy`  
  Distance of the flock center from the goal over time

- `boids_trajectories.pkl`  
  Position trajectories of all agents

---

### `plots.py`
Post-processing and visualization script.

It:
- Loads the simulation outputs generated by the main script
- Produces figures and animations to analyze the system behavior
- Visualizes trajectories, collective metrics, and weight evolution

Outputs:

- `weights_simplex.pdf`  
  Ternary plot of weight evolution for one representative informed agent

- `trajectories.pdf`  
  Plot of the trajectories of all agents

- `flock_metrics_over_time.pdf`  
  Time evolution of:
  - polarization \(p\)
  - milling \(m\)
  - distance to goal \(d\)

- `trajectories_video.mp4`  
  Animation of the flock evolution. Informed agents are shown in red, uninformed agents are shown in blue.

  *Note: playback speed is 2× the simulated dynamics (fps = 2/dt).*

The uninformed-agent models implemented in this folder are based on:

- I. D. Couzin, J. Krause, R. James, G. D. Ruxton, and N. R. Franks,  
  “Collective Memory and Spatial Sorting in Animal Groups,”  
  *Journal of Theoretical Biology* **218**, 1–11 (2002).

- F. Cucker and S. Smale,  
  “Emergent behavior in flocks,”  
  *IEEE Transactions on Automatic Control* **52**, 852–862 (2007).

- T. Vicsek, A. Czirók, E. Ben-Jacob, I. Cohen, and O. Shochet,  
  “Novel type of phase transition in a system of self-driven particles,”  
  *Physical Review Letters* **75**, 1226–1229 (1995).
