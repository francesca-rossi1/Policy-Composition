# Milling

This folder contains the code to replicate the milling experiment from the main text of the paper.

## Contents

### `milling.py`
Main simulation script for this experiment.

It:
- Initializes the agent-based system
- Runs the time evolution of the flock
- Computes collective metrics
- Stores trajectories, weights evolution, and metrics

Outputs:

- `milling_history.npy`  
  Time series of the milling metric

- `polarization_history.npy`  
  Time series of the polarization metric

- `weights_history.npy`  
  Time evolution of the weights for all agents

- `min_distances.npy`  
  Minimum pairwise distance between agents over time

- `boids_trajectories.pkl`  
  Position trajectories of all agents

- `boids_vel_trajectories.pkl`  
  Velocity trajectories of all agents

---
