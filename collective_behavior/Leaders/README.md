# Leaders

This folder contains the code to replicate the leader–follower experiment from the main text of the paper, where a subset of agents (10%) is informed and steers the flock toward a target.

## Contents

### `leaders.py`
Main simulation script for this experiment.

It:
- Initializes a flock with a subset of informed (leader) agents
- Evolves the system dynamics over time
- Computes collective metrics
- Stores trajectories, weights, and metrics

Outputs:

- `weights_history.npy`  
  Time evolution of the interaction weights for all agents

- `polarization_history.npy`  
  Time series of the polarization metric

- `distance_history.npy`  
  Distance between the flock center of mass and the goal over time

- `boids_trajectories.pkl`  
  Position trajectories of all agents

---
