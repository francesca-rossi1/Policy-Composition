# Varying the regularization parameter $\varepsilon$

This folder contains the code to analyze how the regularization parameter $\varepsilon$ influences the collective behavior in the experiments with informed agents.

## Contents

### `varying_eps.py`

Main simulation script for this set of experiments.

It:

* Initializes the agent-based flocking system
* Runs multiple simulations for different values of $\varepsilon$
* Evolves agent weights using the policy composition dynamics
* Computes collective metrics (polarization, milling, distance to goal)
* Stores trajectories of weights and macroscopic observables

Outputs (for each value of $\varepsilon$):

* `polarization_eps{ε}.npy`
  Polarization time series over all simulations

* `milling_eps{ε}.npy`
  Milling metric time series over all simulations

* `distance_eps{ε}.npy`
  Distance of the flock center from the goal over time

* `weights_eps{ε}.npy`
  Time evolution of the policy weights for all agents

Each file has shape `(N_simulations, max_steps, ...)`, enabling statistical analysis across runs.

---

### `plots.py`

Post-processing and visualization script.

It:

* Loads simulation outputs generated by `varying_eps.py`
* Computes averages and standard deviations across simulations
* Produces comparative plots for different values of $\varepsilon$

Outputs:

* `metrics_multi_eps.pdf`
  Time evolution of:

  * Polarization (p)
  * Distance to goal (d)

## Notes

* The script runs simulations for multiple values of $\varepsilon$ (default: `[0.1, 1, 10]`).
* Each configuration is averaged over multiple independent runs (`N_simulations = 30`).
* The parameter $\varepsilon$ controls the "sharpness" of the softmax, thus affecting how decisively agents select behavioral primitives.
